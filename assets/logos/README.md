# Novita Logo

The Novita mark is a **map pin holding a bowl of food** with a **saffron leaf garnish** — location + fresh food in one symbol. It's drawn from the Novita design system (basil green primary, saffron accent).

## Files

| File | Use |
|------|-----|
| `novita-icon.svg` | App icon — full mark on the deep-basil rounded tile (app stores, launchers, favicons). |
| `novita-mark-light.svg` | Bare mark in basil for light / pale-green surfaces. |
| `novita-mark-reverse.svg` | Bare mark in paper-white for dark backgrounds. |
| `novita-mark-mono.svg` | Single-ink silhouette (bowl + leaf knocked out) for one-color print, embossing, etched use. |
| `novita-lockup-horizontal.svg` | Mark + "Novita" wordmark with the saffron dot. Primary lockup. |

## Colors

- Basil 700 `#1B4332` / Basil 900 `#12281C` — the pin
- Paper `#F4F6F1` — knockouts / reverse
- Saffron 500 `#E9A23B` — the leaf garnish and the wordmark dot (used sparingly)

## Wordmark

Set in **Bricolage Grotesque** (800 weight, -4% tracking), matching the design system's display face. The trailing dot is Saffron 500. The lockup SVG references the web font; for print or environments without it, convert the text to outlines.

## Clear space & minimum size

- Keep clear space of at least the bowl's diameter around the mark.
- Minimum icon size 16px; below that use the app-icon tile version for legibility.
